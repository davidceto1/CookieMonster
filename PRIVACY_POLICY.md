# Privacy Policy for CookieMonster

**Last updated:** March 12, 2026

## Data Collection

CookieMonster does **not** collect, transmit, or store any personal data. All data used by the extension stays entirely on your local machine and is never sent to any external server or third party.

## Permissions

CookieMonster requires the following browser permissions to function:

| Permission | Why it's needed |
|---|---|
| `cookies` | Required to set and remove test cookies on specified domains. This is the core functionality of the extension. |
| `storage` | Used to persist your cookie configurations locally within the browser. |
| `tabs` | Used to access the active tab's URL so cookies can be displayed and managed in context. |
| Host permissions (`<all_urls>`) | Required so the extension can set cookies on any domain you configure for testing, not just a fixed set of sites. |

## Third-Party Services

CookieMonster does not use any third-party services, analytics, or tracking.

## Changes to This Policy

If this privacy policy is updated, the changes will be reflected in this document with an updated date.
